# Jupter 가상환경 만들기

##### conda create -n tensor_ai python=3.8

##### conda env list        // 리스트 확인

##### conda activate 생성할폴더이름     // 재접속시 이 코드만 실행하면 된다
##### conda activate tensor_ai

##### pip install tensorflow==2.2

##### conda install jupyter



### 파이썬 패키지 설치
